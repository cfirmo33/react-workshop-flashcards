# Flashcards

## Co to má umět

- Má editační a tréninkový mód
- Editační mód:
  - Mohu přidávat otázky a odpovědi
- Tréninkový mód:
  - Zobrazí kartičky
  - Na kliknutí lze vidět odpověď
  - Mezi kartičkami se lze přepínat
